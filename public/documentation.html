GET /movies/genre/:genreName

Returns a description for a chosen genre of movie.

Example Request:

GET http://yourdomain.com/movies/genre/Drama

Example Response:

Status Code: 200 OK

    {
        "Name": "Drama",
        "Description": "A serious narrative work that presents a realistic portrayal of characters within a social or emotional context."
    }

GET /movies/director/:directorName

Returns data on a chosen director.

Example Request:

GET http://yourdomain.com/movies/director/Christopher%20Nolan

Example Response:

Status Code: 200 OK

    {
        "Name": "Christopher Nolan",
        "Bio": "Christopher Nolan is a British-American film director, producer, and screenwriter. He is known for his work on films such as Memento, The Dark Knight Trilogy, and Inception.",
        "Birth": 1970
    }

PUT /users/:id/:movieTitle

Adds a movie to a user's favorite movies array.

Example Request:

PUT http://yourdomain.com/users/1/The%20Godfather%20Part%20II

Example Response:

Status Code: 200 OK

The response will include a message indicating that the movie has been added to the user's favorite movies array.

PUT /users/:id

Updates a users name by their ID.

Example Request:

PUT http://yourdomain.com/users/1

Request Body:

    {
        "name": "[UPDATED NAME]",
        "favoriteMovies": ["The Godfather", "The Shawshank Redemption", "The Dark Knight"]
    }

Example Response:

Status Code: 200 OK

The response will include the updated user object.

DELETE /users/:id/:movieTitle

Removes a movie from a user's favorite movies array.

Example Request:

DELETE http://yourdomain.com/users/1/The%20Godfather

Example Response:

Status Code: 200 OK

The response will include a message indicating that the movie has been removed from the user's favorite movies array.

DELETE /users/:id

Deletes a user by their ID.

Example Request:

DELETE http://yourdomain.com/users/1

Example Response:

Status Code: 200 OK

The response will include a message indicating that the user has been deleted.

POST /users

Creates a new user.

Example Request:

POST http://yourdomain.com/users

Request Body:

    {
        "name": "Bob Smith",
        "favoriteMovies": ["The Matrix", "Pulp Fiction"]
    }

Example Response:

Status Code: 201 Created

The response will include the newly created user object.
