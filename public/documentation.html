GET /genres/:genreName

Returns a description for a chosen genre of movie.

Example Request:

GET http://yourdomain.com/genre/Thriller

Example Response:

POST /users/:username/movies/:movieID

Adds a movie to a user's favorite movies array.

Example Request:

PUT http://yourdomain.com/users/exampleUser/movies/643352e75683ef91921d2e85

Example Response:

        {
            "_id": "643f561267c7e740fc3fd639",
            "Username": "exampleUser",
            "Password": "examplePass",
            "Email": "example@test.com",
            "Birthday": "2023-10-12T06:00:00.000Z",
            "FavoriteMovies": [
                "643352e75683ef91921d2e85"
            ],
            "__v": 0
        }

Status Code: 200 OK

The response will include a message indicating that the movie has been added to the user's favorite movies array.

PUT /users/:id

Updates a users name by their ID.

Example Request:

PUT http://yourdomain.com/users/1

Request Body:

    {
        "name": "[UPDATED NAME]",
        "favoriteMovies": ["The Godfather", "The Shawshank Redemption", "The Dark Knight"]
    }

Example Response:

Status Code: 200 OK

The response will include the updated user object.

DELETE /users/:id/:movieID

Removes a movie from a user's favorite movies array.

Example Request:

DELETE http://yourdomain.com/users/exampleUser/movies/643352e75683ef91921d2e85

Example Response:

        {
            "_id": "643f561267c7e740fc3fd639",
            "Username": "exampleUser",
            "Password": "examplePass",
            "Email": "example@test.com",
            "Birthday": "2023-10-12T06:00:00.000Z",
            "FavoriteMovies": [],
            "__v": 0
        }

Status Code: 200 OK

The response will include a message indicating that the movie has been removed from the user's favorite movies array.

DELETE /users/:username

Deletes a user by their ID.

Example Request:

DELETE http://yourdomain.com/users/exampleUser

Example Response:

exampleUser was deleted.

Status Code: 200 OK

The response will include a message indicating that the user has been deleted.

POST /users

Creates a new user.

Example Request:

POST http://yourdomain.com/users

Request Body:

        {
            "Username": "exampleUser",
            "Password": "examplePass",
            "Email": "example@test.com",
            "Birthday": "10/12/23"
        }

Example Response:

        {
            "Username": "exampleUser",
            "Password": "examplePass",
            "Email": "example@test.com",
            "Birthday": "2023-10-12T06:00:00.000Z",
            "FavoriteMovies": [],
            "_id": "643f561267c7e740fc3fd639",
            "__v": 0
        }

The response will include the newly created user object.
